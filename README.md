# SpringBootCalculator

## Description

The DebugDataBandits have created a calculator in Spring Boot, which has been deployed. There are both simple & advanced functions of the calculator, including:
- Addition
- Subtraction
- Multiplication
- Division
- Fibonacci
- Factorial
- Square Root
- Power
- Clear
